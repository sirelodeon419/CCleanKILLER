Open Source Windows Utility

Remove CCleaner and bundled Windows bloatware cleanly.

CCleanKILLER is a Windows removal tool that detects and removes CCleaner, Piriform tools, Avast, AVG, scheduled tasks, registry keys, startup entries, and leftover services without exposing an HTTP surface.

What it removes

The scanner targets common CCleaner and Piriform leftovers across files, uninstall traces, services, scheduled tasks, and registry hives.

  • CCleaner, update services, and performance optimizer traces
  • CCleaner Browser, CCleaner Cloud, Avast Free Antivirus, AVG Antivirus
  • Avast Secure Browser, AVG Secure Browser, Recuva, Defraggler, Speccy
  • Telemetry remnants, startup entries, and scheduled tasks
Removing security software is a privileged operation. Always review detections and create a restore point before making system changes.

Why it exists

CCleanKILLER focuses on complete removal, not cosmetic uninstall flows. It targets leftover registry keys, services, and startup entries that normal uninstallers often leave behind.

How it works

Electron + React drives the UI. The main process spawns PowerShell directly and streams scan and removal output over IPC with no local web server involved.

Availability

Current release targets Windows 10 and Windows 11. Portable and NSIS builds are published through GitHub Releases. v2.0.0

Get started

Download the latest build from GitHub Releases, run the executable as Administrator, review detections, and remove flagged components. Source code, license, and rules are available in the public repository.
